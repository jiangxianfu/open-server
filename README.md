# open-server

A simple, lightweight HTTP server built with Node.js.

## Installation

No dependencies required - uses only Node.js built-in modules.

## Usage

Start the server:
```bash
npm start
```

The server will start on port 3000 by default. You can customize the port and host using environment variables:

```bash
PORT=8080 HOST=localhost npm start
```

## Testing

Run the test suite:
```bash
npm test
```

## Features

- Simple HTTP server
- Configurable port and host
- Zero external dependencies
- Built with Node.js built-in `http` module
